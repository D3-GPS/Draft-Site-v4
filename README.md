# D3 Landing Page V4 - Deploy Ready

## What this is
A standalone Next.js review build for D3 Landing Page V4.

## Deploy
1. Unzip the folder
2. Upload to a new GitHub repo
3. Import the repo into Vercel
4. Deploy

## Notes
- This build is standalone and does not require shadcn/ui
- Route targets are placeholder links for Stripe, Tally, and Calendly
- This is a review build, not a production-integrated build
